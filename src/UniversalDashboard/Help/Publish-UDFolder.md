# Publish-UDFolder

## SYNOPSIS
Publishes a folder that will be available to download assets. 

## SYNTAX

```
Publish-UDFolder [-Path] <String> [-RequestPath] <String> [<CommonParameters>]
```

## DESCRIPTION
Publishes a folder that will be available to download assets. 

## EXAMPLES

### Example 1
```
PS C:\> Publish-UDFolder
```
