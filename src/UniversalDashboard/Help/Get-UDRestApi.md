# Get-UDRestApi

## SYNOPSIS
Returns all running REST API servers.

## SYNTAX

```
Get-UDRestApi [-Name <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns all running REST API servers.

## EXAMPLES

### Example 1
```
PS C:\> Get-UDRestApi -Name "RestApi1"
```

Returns the REST API server named RestApi1.

## PARAMETERS

### -Name
The name of the REST API server to return.
