# New-UDIFrame

## SYNOPSIS
Creates a new iframe.

## SYNTAX

```
New-UDIFrame [[-Id] <String>] [[-Uri] <Object>] [<CommonParameters>]
```

## DESCRIPTION
Creates a new iframe.

## EXAMPLES

### Example 1
```
PS C:\> New-UDIFrame -Uri https://www.google.com
```

Creates an iframe that embeds Google.
