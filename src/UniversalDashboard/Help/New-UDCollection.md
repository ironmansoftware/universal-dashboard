# New-UDCollection

## SYNOPSIS
Creates a collection of items.

## SYNTAX

```
New-UDCollection [[-Id] <String>] [[-Content] <ScriptBlock>] [-LinkCollection] [[-Header] <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Creates a collection of items.

## EXAMPLES

### Example 1
```
PS C:\> New-UDCollection -Content {
    New-UDCollectionItem -Content { 
        "Item 1"
    }
    New-UDCollectionItem -Content { 
        "Item 2"
    }
    New-UDCollectionItem -Content { 
        "Item 3"
    }
}
```

Creates a collection of 3 items.
