# Select-UDElement

## SYNOPSIS
Selects an element. 

## SYNTAX

```
Select-UDElement -ID <String> [<CommonParameters>]
```

## DESCRIPTION
Selects an element. 

## EXAMPLES

### Example 1
```
PS C:\> Select-UDElement -Id 'textbox'
```

Selects the text box with the ID 'textbox'

## PARAMETERS
